Luna OS
Windows의 접근성, macOS의 미학, Linux의 UNIX 철학.

Simple enough to understand. Powerful enough to create.

Vision
Luna OS는 아름다움과 기능이 완벽하게 공존하는 차세대 Linux 기반 데스크톱 운영체제이다. 우리는 복잡함을 숨기고, 가능성을 드러낸다.

Mission
컴퓨터를 “배워야 하는 도구”가 아닌, “바로 몰입할 수 있는 공간”으로 만든다. 사용자는 시스템을 이해하기 위해 시간을 낭비하지 않는다. 오직 창조하는 데 집중한다.

Core Principles
Clarity over complexity
사용자는 길을 찾지 않는다. 시스템이 길을 보여준다.

Consistency is trust
모든 인터페이스는 하나의 언어로 말한다. 익숙함은 학습이 아닌 경험에서 온다.

UNIX philosophy, modernized
작게 만들고, 정교하게 조합한다. 단순함은 설계의 완성도이다.

Performance is respect
빠름은 단순한 속도가 아니다. 사용자의 시간을 존중하는 배려이다.

Design for everyone
강력함은 선택된 사람만의 것이 아니다. 누구나 누릴 수 있는 표준이다.

Promise
Luna OS는 사용자를 가르치지 않는다. 사용자가 원하는 것을 즉시 가능하게 만든다.
